# Kavi Labs Kanban

Drop tasks, files, text, audio — anywhere. The agent picks them up.

## Features

- **Drag and drop** cards between columns (mouse + touch)
- **Drop files anywhere** on the page to create a task instantly
- **Voice memos** — record audio right in the task modal
- **Any file type** — images, PDFs, docs, audio, video
- **Mobile-first** — works perfectly on phone (Will optimized)
- **Keyboard shortcuts** — `N` new task, `R` record, `Esc` close
- **Context menus** — right-click (or long-press) to edit/move/delete
- **Local storage** persistence — no backend needed

## Usage

Just open `index.html` in any browser. No build step, no server required.

## Columns

Default columns: Ideas → To Do → In Progress → Done

Add/edit/delete columns freely. Color-coded dots for each column.

## Shortcuts

| Key | Action |
|-----|--------|
| `N` | New task |
| `R` | Start/stop voice recording |
| `Esc` | Close modal |
| Right-click | Context menu |
| Long-press (mobile) | Context menu |

## Stack

Pure HTML/CSS/JS — no frameworks, no dependencies, zero build step.
